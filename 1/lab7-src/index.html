Part 5. Implementing Text to Speech

Write a program

  ./TextToSpeech phrase wavefile
 
Example:

  ./TextToSpeech "Hello world" hello.wav

that will generate the speech that corresponds to the phrase "Hello world" using your own voice and it will write it in the file hello.wav.

You will generate the text to speech as follows:
  • Download the program "Audacity" from audacity.sourceforge.com
  • Record with your own voice each of the folowing words and name the file <word>.wav. Save them as WAV 16bit signed. Record just a few words initially to test your program. Record the rest when you are sure your program works correctly. Save them using the option File->Export->wave 16bit signed.
        time, good, to, the, person, have, new, of, and, year, do, first, in, a, way, say, last, for, that, day, get,
        long, on, I, thing, make, great, with, it, man, world, life, hand, part, child, eye, woman, place, work, week.
  • Parse the phrase passed as argument into words. Discard all the punctuation marks and change it to lower case. Using the Sound class you wrote, read the corresponding wav files for these words and append them into a single sound. Write the resulting sound into the wavefile passed as argument. If the word does not have a corresponding sound file skip it. Write a silence of about .05secs between each word.
You will show your texttospeech program during the lab.
